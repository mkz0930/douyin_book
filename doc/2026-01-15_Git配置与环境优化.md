# Git 配置与环境优化 (2026-01-15)

## 1. 变更背景
为了防止生成的 MP4 视频文件（通常较大且为临时/结果产物）被误提交到 Git 仓库，需要更新 Git 忽略规则。同时，为了更好地管理项目文档，建立专门的文档目录。

## 2. 详细变更
### 2.1 .gitignore 更新
在 `.gitignore` 文件中添加了以下规则：
```gitignore
*.mp4
```
这确保了根目录及子目录下的所有 `.mp4` 文件都会被 Git 忽略。

### 2.2 文档目录结构
创建了 `doc/` 目录。建议后续所有的功能文档、更新日志、测试报告都存放在此目录下，保持项目根目录整洁。

## 3. 验证
- 检查 `.gitignore` 文件内容，确认包含 `*.mp4`。
- 确认 `doc/` 目录已存在。
